# Zentity-Task

This repository contains a Postman collection for interacting with Veronika's Books API.

## Endpoints:

### Get All Books
- **GET** `/books/{{bookId}}`

### Get Book by ID
- **GET** `/books/{{bookId}}`

### Get Book by Author Name
- **GET** `/books?author={{authorFirstName}}%20{{authorLastName}}`

### Create Book
- **POST** `/books

### Update Book
- **PUT** `/books/{{bookId}}

### Delete Book
- **DELETE** `/books/{{bookId}}
